# 📦 FotoCub 3D

A 3D rotating cube for Android with custom photos on each face, built with OpenGL ES.

## ✨ Features

- 🖼️ 6 custom photos — one on each face of the cube
- 🔄 Auto-rotation with adjustable speed
- 🖐️ Touch to rotate manually in any direction
- 🔍 Pinch to zoom in / out
- ↩️ Double-tap to reset position
- ⏸️ Pause / Play button
- 📷 Save screenshot to gallery
- 💡 3D lighting for a realistic look

---

## 🛠️ Built With

- Java
- Android OpenGL ES 1.0
- Android Studio

---

## 📁 How to add your photos

Place 6 images named `pic1.jpg` to `pic6.jpg` inside `res/drawable/`.  
Square images work best (e.g. 512×512 or 1024×1024 px).
